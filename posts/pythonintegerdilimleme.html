Python ile Integer Veri Dilimleme



Python ile TC Kimlik No üretir uygulamamda integer veri dilimlemem gerekiyordu çünkü TC Kimlik No algoritmasında 11 tane rakamın ilk 10 tanesinin toplamının birler basamağı, 11. rakama eşit olması gerekmekteydim.

String veriyi dilimlemeyi biliyordum. Ben de düşündüm ki integer veriyi string veriye dönüştürürsem sonra istediğim kadar dilimleyip tekrar integer değere dönüştürebilirdim.
